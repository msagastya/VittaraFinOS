## Batch A — Wire Voice + OCR to Actual Saves
**Priority: Critical — without this, Phase 5 is dead weight**
**Estimated files changed: 6–8**

### A1. Voice → Transaction Save
File: `lib/ui/voice/voice_overlay_widget.dart`
File: `lib/ui/dashboard/quick_entry_sheet.dart`

- `VoiceIntent.addExpense / addIncome` → build `Transaction` from fields, call
  `TransactionsController.addTransaction()`, call `TransactionAccountAdjuster.applyTransaction()`

- `VoiceIntent.addInvestment` → build `Transaction(type: investment)` + `Investment` record
  if investment controller is accessible
- `VoiceIntent.navigate` → call `VoiceNavigator.resolve()` → push the correct screen via
  the global navigator key

  speak it via `VoiceController._speak()`

Wire: `VoiceOverlayWidget.show()` returns the completed `VoiceResult`, caller handles save.
Add a `GlobalNavigatorKey` singleton so voice can navigate from anywhere.

## Batch C — Hardcoded Color Cleanup
**Priority: High — technical debt, affects maintainability**
**Strategy: Incremental, screen by screen, not a big-bang rewrite**

### C1. Lint rule to flag new violations
File: `analysis_options.yaml`

Add custom lint to flag raw `Color(0xFF...)` outside `app_styles.dart` and `design_tokens.dart`.
This stops the bleed while we clean up.

### C2. Map all hardcoded colors → AppStyles tokens

Common offenders found (2,390+ instances):
- `Color(0xFF000000)` → `AppStyles.getBackgroundColor(context)`
- `Color(0xFF0D0D0D)` → `AppStyles.getCardColor(context)`
- `Color(0xFF1C1C1C)` → `AppStyles.getDividerColor(context)` (add if missing)
- `Color(0xFFFFFFFF)` → `CupertinoColors.white`
- `Color(0xFF4A90D9)` → add `AppStyles.accentBlue` token
- `Color(0xFF6C63FF)` → add `AppStyles.investmentPurple` token
- `Color(0xFFFF6B6B)` → `AppStyles.loss(context)`
- `Color(0xFF4CAF50)` → `AppStyles.gain(context)`

### C3. Cleanup order (by screen complexity, easiest first)
1. `lib/ui/styles/` files themselves — ensure all tokens exist
2. `lib/ui/dashboard/widgets/` — 8 widget files
3. `lib/ui/manage/` — investments, goals, budgets screens
4. `lib/logic/ai/` — new AI UI files (just written, easiest to fix now)
5. `lib/ui/transaction_history_screen.dart` — largest single file
6. Everything else — background sweep
