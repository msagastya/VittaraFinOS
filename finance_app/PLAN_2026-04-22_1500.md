## Core Principles

1. **No internet except investment price feeds.** Every network call is blocked at the Android OS level except a whitelist of investment data domains. Zero exceptions.
2. **On-device AI only.** All intelligence runs locally. No API keys, no cloud, no telemetry.
3. **Encrypted-at-rest.** Every byte of user data is AES-256 encrypted on disk.
4. **Fail-safe writes.** Every data write is atomic, mutex-protected, and shows the user an error if it fails.
5. **Best free models.** Three AI tiers: Gemma 2B (LLM), ML Kit Entity Extraction, TFLite classifier — each with automatic fallback to the tier below.

## 1B. Code-Level Enforcement — `SecureNetworkClient`

OS-level blocking is the primary defense. Code-level is the secondary layer that makes it obvious to any future developer (including yourself) which code is allowed to call out.

**New file: `lib/services/network/secure_network_client.dart`**

```dart
/// The ONLY class in VittaraFinOS that may make outbound HTTP calls.
///
/// Every call passes through [_InvestmentDomainGuard] which verifies
/// the target domain is on the investment whitelist before sending.
///
/// All other code must NOT import 'package:http/http.dart' directly.
/// The linter rule in analysis_options.yaml enforces this.
library secure_network_client;

import 'dart:convert';
import 'package:http/http.dart' as http;

// Domains allowed to receive network requests from this app.
const _kAllowedDomains = {
  'query1.finance.yahoo.com',
  'query2.finance.yahoo.com',
  'finance.yahoo.com',
  'www.amfiindia.com',
  'api.mfapi.in',
  'open.er-api.com',
  'api.exchangerate-api.com',
  'www.nseindia.com',
  'api.coingecko.com',
};

class SecureNetworkClient {
  static final SecureNetworkClient _instance = SecureNetworkClient._();
  factory SecureNetworkClient() => _instance;
  SecureNetworkClient._();

  final _client = http.Client();

  Future<Map<String, dynamic>> getJson(Uri uri, {Map<String, String>? headers}) async {
    _guard(uri);
    final response = await _client.get(uri, headers: headers)
        .timeout(const Duration(seconds: 10));
    if (response.statusCode != 200) {
      throw NetworkException('HTTP ${response.statusCode} from ${uri.host}');
    }
    return jsonDecode(response.body) as Map<String, dynamic>;
  }

  Future<String> getText(Uri uri, {Map<String, String>? headers}) async {
    _guard(uri);
    final response = await _client.get(uri, headers: headers)
        .timeout(const Duration(seconds: 15));
    if (response.statusCode != 200) {
      throw NetworkException('HTTP ${response.statusCode} from ${uri.host}');
    }
    return response.body;
  }

  void _guard(Uri uri) {
    if (!_kAllowedDomains.contains(uri.host)) {
      throw NetworkException(
          'SECURITY: Network call to "${uri.host}" blocked. '
          'Only investment price domains are permitted.');
    }
  }
}

class NetworkException implements Exception {
  final String message;
  const NetworkException(this.message);
  @override
  String toString() => 'NetworkException: $message';
}
```

## 1C. Encrypted SQLite Storage

**Why:** SharedPreferences and plain SQLite files are readable by any forensic tool, rooted device, or backup extraction. A financial app must encrypt all transaction, account, and personal data at rest.

**Implementation:**

`sqflite` is already in pubspec. We use it with a per-device encryption key stored in Android Keystore via `flutter_secure_storage` (also already in pubspec).

Key derivation:
1. On first launch, generate a 256-bit random key via `dart:math Random.secure()`
2. Store it in `flutter_secure_storage` → backed by Android Keystore (hardware-backed on modern devices)
3. For every DB open, retrieve the key and pass it to SQLite's `PRAGMA key`
4. Use `sqflite_sqlcipher` package which bundles SQLCipher (AES-256-CBC)

```yaml
# pubspec.yaml addition
sqflite_sqlcipher: ^2.2.1+1   # replaces sqflite for encrypted DB
```

Note: `sqflite` and `sqflite_sqlcipher` share the same API. Migration is a drop-in replacement.

**New file: `lib/services/database_service.dart`**

```dart
class DatabaseService {
  static final DatabaseService instance = DatabaseService._();
  DatabaseService._();

  Database? _db;
  Database get db => _db!;

  Future<void> open() async {
    // 1. Get or generate encryption key
    const storage = FlutterSecureStorage(
      aOptions: AndroidOptions(encryptedSharedPreferences: true),
    );
    String? key = await storage.read(key: 'db_encryption_key');
    if (key == null) {
      final rand = Random.secure();
      final bytes = List.generate(32, (_) => rand.nextInt(256));
      key = base64Url.encode(bytes);
      await storage.write(key: 'db_encryption_key', value: key);
    }

    // 2. Open encrypted SQLite DB
    final docDir = await getApplicationDocumentsDirectory();
    _db = await openDatabaseWithOptions(
      join(docDir.path, 'vittara.db'),
      password: key,      // SQLCipher AES-256
      version: 1,
      onCreate: _onCreate,
      onUpgrade: _onUpgrade,
    );
  }

  Future<void> _onCreate(Database db, int version) async {
    await db.execute('''CREATE TABLE transactions (
      id TEXT PRIMARY KEY,
      description TEXT NOT NULL,
      amount REAL NOT NULL,
      type INTEGER NOT NULL,

      is_archived INTEGER NOT NULL DEFAULT 0,
      created_at INTEGER NOT NULL
    )''');

    await db.execute('''CREATE TABLE accounts (
      id TEXT PRIMARY KEY,
      name TEXT NOT NULL,
      bank_name TEXT NOT NULL,
      type INTEGER NOT NULL,
      balance REAL NOT NULL DEFAULT 0,
      currency TEXT NOT NULL DEFAULT 'INR',
      color_value INTEGER,
      icon_code INTEGER,
      is_active INTEGER NOT NULL DEFAULT 1,
      notes TEXT,
      created_at INTEGER NOT NULL
    )''');

    await db.execute('''CREATE TABLE investments (
      id TEXT PRIMARY KEY,

      type INTEGER NOT NULL,
      amount REAL NOT NULL,
      current_value REAL,
      broker TEXT,
      notes TEXT,
      purchase_date INTEGER,
      color_value INTEGER,

      created_at INTEGER NOT NULL
    )''');

    await db.execute('''CREATE TABLE goals (
      id TEXT PRIMARY KEY,
      name TEXT NOT NULL,
      target_amount REAL NOT NULL,
      current_amount REAL NOT NULL DEFAULT 0,
      deadline INTEGER,
      icon_code INTEGER,
      color_value INTEGER,
      notes TEXT,
      created_at INTEGER NOT NULL
    )''');

    await db.execute('''CREATE TABLE budgets (
      id TEXT PRIMARY KEY,
      name TEXT NOT NULL,
      limit_amount REAL NOT NULL,
      spent_amount REAL NOT NULL DEFAULT 0,
      category_name TEXT,
      period INTEGER NOT NULL,
      start_date INTEGER NOT NULL,
      color_value INTEGER,
      created_at INTEGER NOT NULL

    // Indexes for fast queries
    await db.execute('CREATE INDEX idx_tx_dt ON transactions(date_time DESC)');
    await db.execute('CREATE INDEX idx_tx_type ON transactions(type)');
    await db.execute('CREATE INDEX idx_tx_acc ON transactions(source_account_id)');
    await db.execute('CREATE INDEX idx_tx_arch ON transactions(is_archived)');
  }

  // One-time migration from SharedPreferences JSON
  Future<void> migrateFromSharedPrefsIfNeeded() async {
    final prefs = await SharedPreferences.getInstance();
    await _migrateTable(prefs, 'transactions', 'transactions');
    await _migrateTable(prefs, 'accounts', 'accounts');
    await _migrateTable(prefs, 'investments', 'investments');
    await _migrateTable(prefs, 'goals', 'goals');
    await _migrateTable(prefs, 'budgets', 'budgets');
    // Archive was stored separately — merge into transactions with is_archived=1
    final archJson = prefs.getStringList('transactions_archive') ?? [];
    if (archJson.isNotEmpty) {
      final existing = await db.query('transactions',
          where: 'is_archived = 1', limit: 1);
      if (existing.isEmpty) {
        final batch = db.batch();
        for (final row in archJson) {
          final map = jsonDecode(row) as Map<String, dynamic>;
          map['is_archived'] = 1;
          batch.insert('transactions', map,
              conflictAlgorithm: ConflictAlgorithm.ignore);
        }
        await batch.commit(noResult: true);
        await prefs.remove('transactions_archive');
      }
    }
  }

  Future<void> _migrateTable(
      SharedPreferences prefs, String prefsKey, String table) async {
    final rows = prefs.getStringList(prefsKey);
    if (rows == null || rows.isEmpty) return;
    final existing = await db.query(table, limit: 1);
    if (existing.isNotEmpty) return; // already migrated
    final batch = db.batch();
    for (final row in rows) {
      try {
        final map = jsonDecode(row) as Map<String, dynamic>;
        batch.insert(table, map, conflictAlgorithm: ConflictAlgorithm.ignore);
      } catch (_) {}
    }
    await batch.commit(noResult: true);
    await prefs.remove(prefsKey);
  }
}
```

**IMPORTANT — Gemma model download and network policy:**

The Gemma model binary is 1.5 GB. We have two options:
- **Option A:** Add `storage.googleapis.com` to `network_security_config.xml` and build a one-time download flow inside the app
- **Option B:** Use MediaPipe's `LlmInference.createFromModelPath()` with the model bundled via a separate download tool, bypassing our network config

**Recommended: Option A** — Add `storage.googleapis.com` to the network config ONLY in a `debug` or `setup` config. The `network_security_config.xml` supports `debug-overrides` exactly for this purpose:

```xml
<debug-overrides>
  <!-- Only active in debug builds — allows model download in development -->
  <domain-config cleartextTrafficPermitted="false">
    <domain includeSubdomains="true">storage.googleapis.com</domain>
  </domain-config>
</debug-overrides>
```

For production: user taps "Enable AI" → we show a file picker → they download the model file externally and import it. This keeps the production network config 100% clean.

**New file: `lib/logic/search/nl_search_engine.dart`**

```dart
class NLSearchEngine {
  static final _extractor =
      EntityExtractor(language: EntityExtractorLanguage.english);

  /// Returns results as a stream.
  /// First value = Pass 1 (rule-based, instant, <1ms)
  /// Second value = Pass 2 (ML Kit enhanced, ~80ms on first call, ~10ms cached)
  /// Third value = Pass 3 (Gemma, ~500ms, only if downloaded)
  static Stream<ParsedSearchQuery> parse(String raw) async* {
    // Pass 1 — instant rule-based
    final p1 = RuleBasedParser.parse(raw);
    if (p1 != null) yield p1;

    // Pass 2 — ML Kit
    try {
      final annotations = await _extractor.annotateText(raw)
          .timeout(const Duration(milliseconds: 300));
      final p2 = _enhanceWithMlKit(p1, annotations, raw);
      if (p2 != null) yield p2;
    } catch (_) {}

    // Pass 3 — Gemma (if ready)
    if (GemmaService.instance.isReady) {
      try {
        final structured = await GemmaService.instance.parseSearchQuery(raw);
        if (structured != null) yield ParsedSearchQuery.fromGemmaJson(structured, raw);
      } catch (_) {}
    }
  }

  static ParsedSearchQuery? _enhanceWithMlKit(
      ParsedSearchQuery? base, List<EntityAnnotation> annotations, String raw) {
    double? mlAmount;
    DateTime? mlDateFrom;
    DateTime? mlDateTo;

    for (final ann in annotations) {
      for (final entity in ann.entities) {
        if (entity is MoneyEntity && mlAmount == null) {
          mlAmount = entity.integerPart.toDouble() +
              (entity.fractionalPart / 100.0);
          // ML Kit returns in original currency — convert if needed
          if (entity.unnormalizedCurrency?.toLowerCase() != 'inr') {
            // Leave amount as-is; rule-based already handles ₹/Rs
          }
        }
        if (entity is DateTimeEntity) {
          final range = _dateEntityToRange(entity);
          mlDateFrom = range.$1;
          mlDateTo = range.$2;
        }
      }
    }

    // Only update if ML Kit found something rule-based missed
    if (mlAmount == null && mlDateFrom == null) return null;
    return ParsedSearchQuery(
      displaySummary: base?.displaySummary ?? 'AI filtered',
      keyword: base?.keyword,
      fromDate: mlDateFrom ?? base?.fromDate,
      toDate: mlDateTo ?? base?.toDate,
      type: base?.type,
      minAmount: mlAmount != null && base?.minAmount == null ? mlAmount : base?.minAmount,
      maxAmount: base?.maxAmount,
      categoryHints: base?.categoryHints ?? [],
    );
  }
}
```

## 3C. TFLite Merchant → Category Classifier

**Package:** `tflite_flutter: ^0.10.4`

**Model:** A custom-trained text embedding similarity model or a pre-trained MobileNet variant adapted for text classification.

**What it does:** Given a merchant name ("Dominos", "Apollo Pharmacy", "IRCTC"), predicts the most likely category with a confidence score.

**Training data:** The `_catMap` in `_NLQueryParser` gives us 11 categories × 15+ seed keywords each — that's 165+ labeled training examples. We expand with public merchant category datasets and train a 4-layer MLP on top of character n-gram embeddings.

**Architecture (lightweight, <2MB model):**
- Input: character 3-gram bag-of-words (vocabulary size ~500)
- Hidden: 64 → 32 neurons, ReLU
- Output: 11 category softmax
- Size: ~500KB
- Inference: <5ms on CPU

**New file: `lib/logic/ai/merchant_classifier.dart`**

```dart
class MerchantClassifier {
  static final MerchantClassifier instance = MerchantClassifier._();
  MerchantClassifier._();

  Interpreter? _interpreter;

  Future<void> load() async {
    _interpreter = await Interpreter.fromAsset('assets/models/merchant_classifier.tflite');
  }

  /// Returns (categoryName, confidence) or null if below threshold.
  (String, double)? classify(String merchantName) {
    if (_interpreter == null) return null;
    final input = _encode(merchantName);
    final output = List.filled(11, 0.0).reshape([1, 11]);
    _interpreter!.run([input], output);
    final scores = output[0] as List<double>;
    final maxIdx = scores.indexWhere((s) => s == scores.reduce(math.max));
    final confidence = scores[maxIdx];
    if (confidence < 0.65) return null; // below threshold → don't override
    return (_kCategories[maxIdx], confidence);
  }

  static const _kCategories = [
    'Food', 'Groceries', 'Transport', 'Shopping', 'Entertainment',
    'Health', 'Utilities', 'Rent', 'Insurance', 'Investment', 'Subscription',
  ];
}
```

**Where it's used:**
1. Voice entry — after parsing merchant name, auto-fill category if classifier is confident
2. SMS auto-scan — improve category assignment from merchant name
3. Manual transaction entry — suggest category chip as user types merchant name
4. CSV import — categorize imported rows that have no category

# Phase 4 — Error Handling & Data Safety

## 4A. SafeStorageMixin

**New file: `lib/utils/safe_storage_mixin.dart`**

```dart
mixin SafeStorageMixin on ChangeNotifier {
  String? _lastSaveError;
  String? get lastSaveError => _lastSaveError;

  Future<T?> safeWrite<T>(
    String operationName,
    Future<T> Function() fn,
  ) async {
    try {
      final result = await fn();
      _lastSaveError = null;
      return result;
    } on DatabaseException catch (e, stack) {
      _lastSaveError = 'Database error: ${e.message}';
      debugPrint('[SafeWrite] $operationName failed: $e\n$stack');
      ToastController().showError('Save failed — $operationName');
      return null;
    } catch (e, stack) {
      _lastSaveError = e.toString();
      debugPrint('[SafeWrite] $operationName failed: $e\n$stack');
      ToastController().showError('Save failed — try again');
      return null;
    }
  }
}
```

Applied to all 5 controllers. Every `_save()` call becomes `safeWrite('save transactions', () => _save())`.
