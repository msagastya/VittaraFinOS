## Batch A — P0: Broken core flows (do first, highest impact)

### A1 — Auto-seed "Cash in Hand" on first install
**File:** `lib/logic/accounts_controller.dart`
- In `loadAccounts()`: after loading from SharedPrefs, if list is empty AND a
  flag `accounts_seeded_v1` does NOT exist in prefs → create one Account:
  `name: 'Cash in Hand', type: AccountType.cash, balance: 0.0, bankName: 'Cash'`
  and save it. Set the flag so it never repeats.
- This means every fresh install automatically has Cash in Hand at ₹0.
- Status: [ ]

### A2 — Wire onboarding card steps 2 & 3
**File:** `lib/ui/dashboard/widgets/onboarding_card_widget.dart`
- `case 'transaction'`: call `showQuickEntrySheet(context)` with branch = expense
- `case 'income'`: call `showQuickEntrySheet(context)` with branch = income
- Auto-mark both steps done via `_syncSteps` (already reads txCtrl.transactions,
  so once a transaction/income is saved the card will tick automatically)
- Status: [ ]

### A3 — Block saving a transaction with no account (require account)
**File:** `lib/ui/dashboard/quick_entry_sheet.dart`
- Add `_selectedAccountId != null` to `_canSave` getter
- When account is null AND user taps Save: show an inline red banner
  "Select an account to save this transaction" (not a toast — inline, sticky)
- Also: if NO accounts exist at all, show a prominent banner at top of sheet:
  "No accounts yet — " + tappable "Add Cash in Hand" link that opens AccountWizard
  pre-selected to Cash type. On wizard completion, auto-select the new account.
- Status: [ ]
