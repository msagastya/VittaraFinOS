## Chapter 1 — Physics Engine Foundation
**Status: PENDING**

### 1A. Create `lib/ui/styles/app_springs.dart`
- Define `AppSprings` class with 5 named `SpringDescription` presets:
  - `crisp`   — mass 1.0, stiffness 400, damping 30  (instant snappy response)
  - `natural` — mass 1.0, stiffness 200, damping 22  (everyday interactions)
  - `bouncy`  — mass 1.0, stiffness 180, damping 12  (delightful overshoots)
  - `gentle`  — mass 1.0, stiffness 120, damping 20  (slow graceful motion)
  - `snappy`  — mass 1.0, stiffness 600, damping 40  (tightest, toolbar-grade)
- All depend only on `package:flutter/physics.dart`

### 1B. Upgrade `BouncyButton` in `lib/ui/widgets/animations.dart`
- Replace `TweenSequence` + fixed 100ms with real `SpringSimulation(AppSprings.crisp, 0, -0.08, 0)`
- `_AnimationController` driven by `animateWith(simulation)` not `.forward(duration:)`
- On tap-down → start spring to −8% scale; on tap-up → spring back to 0%
- Keep haptic: `Haptics.light()` on tap-down
- Preserve the child → builder pattern so all existing callsites keep working

### 1C. Upgrade `CardDeckView` spring return in `lib/ui/widgets/card_deck_view.dart`
- On `_onPanEnd`: if below threshold, use `SpringSimulation(AppSprings.natural, _dragOffset, 0.0, velocityX)` where `velocityX = details.velocity.pixelsPerSecond.dx`
- Replace `Curves.elasticOut` + fixed 400ms with `_returnController.animateWith(simulation)`
- Dot indicators: replace `Theme.of(context).colorScheme.primary` with `SemanticColors.getPrimary(context)`
- Card padding: reduce horizontal from 16 to 10 and add peek offset so next card edge is visible

### 1D. Upgrade `FadeScalePageRoute` in `lib/ui/widgets/animations.dart`
- Replace fixed-300ms `CurvedAnimation` with `AnimationController.animateWith(SpringSimulation(AppSprings.gentle, 0.96, 1.0, 1.2))`
- Scale 0.96 → 1.0 spring-driven; fade still uses linear 200ms curve (separate controller)
- Same for pop: reverse spring on dismiss

## Chapter 2 — Typography Foundation
**Status: PENDING**

### 2A. Fix `lib/ui/styles/typography.dart`
- Remove `fontFamily: systemFont` from all 18 `AppTypography.*` methods
- Remove `fontFamily: displayFont` from all display/hero/largeTitle methods
- Keep `fontFamily: 'monospace'` ONLY in `AppTypography.monospace()`
- This unlocks GoogleFonts (Plus Jakarta Sans / Space Grotesk) set in main.dart
- Validate: no other `fontFamily` references remain except monospace

### 2B. Global scroll physics in `lib/main.dart`
- Wrap `MaterialApp`/`CupertinoApp` child with `ScrollConfiguration(behavior: _AppScrollBehavior())` 
- `_AppScrollBehavior` overrides `getScrollPhysics` to return `BouncingScrollPhysics(parent: AlwaysScrollableScrollPhysics())`
- This gives iOS-style rubber-band to every scrollable in the app with one change
